<h1 align="center">📚 Online Bookstore SQL Project</h1>

<p align="center">
SQL Data Analysis Project
</p>

<hr>

<h2>📌 Project Overview</h2>

<p>
This project analyzes an <b>Online Bookstore Database</b> using SQL queries. 
The goal of the project is to explore customer purchasing behavior, book inventory, and sales performance.
</p>

<p>
Various SQL queries were used to extract insights such as revenue generation, best-selling books, customer spending patterns, and stock availability.
</p>

<hr>

<h2>🗂 Database Tables</h2>

<ul>
<li><b>Books</b> – Contains book details such as title, author, genre, price, and stock.</li>
<li><b>Customers</b> – Contains customer information including name, country, and city.</li>
<li><b>Orders</b> – Stores order transactions including book ID, quantity, order date, and total amount.</li>
</ul>

<hr>

<h2>📊 SQL Analysis Performed</h2>

<ul>
<li>Retrieve books by genre</li>
<li>Find books published after a specific year</li>
<li>Identify customers by location</li>
<li>Analyze orders within a specific time period</li>
<li>Calculate total stock of books</li>
<li>Find the most expensive and lowest stock books</li>
<li>Identify customers placing multiple orders</li>
<li>Calculate total revenue generated from orders</li>
<li>Analyze book sales by genre</li>
<li>Find the most frequently ordered book</li>
<li>Calculate average book price by genre</li>
<li>Identify top spending customers</li>
<li>Calculate remaining stock after fulfilling orders</li>
</ul>
